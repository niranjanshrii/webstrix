# Webtrix Rebuild - Vibe Coding Arena

This repository contains the source code for the "Webtrix Rebuild" competition.

## Challenge
Rebuild a given website design with precision, speed, and clean execution within 90 minutes.

## Stack
- HTML5
- CSS3
- Vanilla JavaScript

## Deployment
This project is designed to be instantly deployable via **Vercel** or **Netlify** using drag-and-drop. Simply drag this entire folder into the dashboard for an instant live URL.
